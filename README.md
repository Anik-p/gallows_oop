
# gallows_OOP

Консольная реализация игры "Виселица" на Python с использованием ООП.

## Особенности

- Система уровней сложности
- Подсчет очков с учетом времени и ошибок
- Визуализация прогресса игры

Игрок пытается отгадать "загаданное" компьютером слово. За один раз можно попытаться отгадать только одну букву. Если буквы была отгадана верно, то она открывается, если нет, игроку засчитывается ошибка. Каждая новая ошибка приближает игрока к проигрышу.

Данные хранятся в виде json файла.
